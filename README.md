# 最高のメンターさんと一緒に作る学習教材💛
**今まで、一番手の込んだWebアプリです(๑>◡<๑)**

1. バックエンドはFirebaseを使用
2. 認証機能はFirebaseAuthを使用
3. CSSのフレームワークは導入せず、UIはCSSで見た目を作成

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
